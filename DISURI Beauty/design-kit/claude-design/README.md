# Shopify UI Kit · DISURI Beauty

A high-fidelity recreation of the disuribeauty.com storefront, built off the written brand guide. Click-thru prototype covering:

- **Home**, editorial hero with bilingual tagline, featured products, science strip
- **Collection**, 3-up product grid with filter chips
- **PDP**, product detail with 4:5 imagery, bilingual ingredients, "Made in Korea" badge
- **Cart**, Porcelain background, Midnight blue promo line

## Components
- `Nav.jsx`, top nav with cart count and language toggle
- `Footer.jsx`, Porcelain footer with bilingual links
- `Hero.jsx`, full-bleed editorial hero with Noir gradient overlay
- `KitProductCard.jsx`, 4:5 imagery, Bodoni name, Frutiger price (kit-internal; the reusable one is the DS `ProductCard`)
- `Button.jsx`, Primary / Secondary / Tertiary variants
- `Badge.jsx`, pill badge for "Made in Korea", "Bestseller", etc.
- `Screens.jsx`, Home / Collection / PDP / Cart compositions

Open `index.html` to drive the prototype.
